## ⚡ Quick Start (30 seconds)

1. Open: https://fyw9unz4.scispace.co
2. Toggle "Instructor View" (top right switch)
3. Click any case in the sidebar to jump to it
4. Click "Reveal Answer" when ready to show analysis

---

## 🎛️ Key Features

### Instructor Toggle (Top Right)
- **OFF (Student View):** Shows only case facts
- **ON (Instructor View):** Shows teaching guidance, Socratic questions, common mistakes

### Progressive Reveal Buttons
- **"Reveal Answer & Analysis"** → Shows complete answer
- **"Hide Answer & Analysis"** → Hides answer again

### Navigation Sidebar (Left)
- Click any case name to jump directly to it
- Color-coded by difficulty level
- Shows time estimates for each case

## 🎯 Classroom Flow

### Option 1: Full Control (Recommended)
1. Project website on main screen
2. Keep instructor toggle ON (students don't see your notes)
3. Navigate through cases using sidebar
4. Reveal answers when ready

### Option 2: Student Devices
1. Share link with students
2. Students work at own pace
3. You facilitate discussion
4. Reveal answers on main screen for group discussion

### Option 3: Hybrid
1. Project on main screen
2. Students also access on devices
3. Students attempt individually
4. Discuss as a group
5. Reveal answers together

## 🆘 Troubleshooting

**Q: Website not loading?**
A: Check internet connection. Website requires internet access.

**Q: Can't see instructor notes?**
A: Toggle the switch in the top right to "Instructor View"

**Q: Navigation not working?**
A: Try refreshing the page (F5 or Cmd+R)

**Q: Need to print?**
A: Use browser print function (Ctrl+P or Cmd+P)
